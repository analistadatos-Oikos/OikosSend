## 📋 Descripción

**OikosSend** es una API profesional diseñada específicamente para **Constructora Oikos** que automatiza el envío de mensajes de WhatsApp personalizados para proyectos inmobiliarios.

### 🏗️ Proyecto Actual: **ALMAX Malambo**

- 📦 Bodegas desde 146.47m² hasta 151.51 m²
- 💰 Precios desde $624 millones
- 📍 Ubicación estratégica cerca al aeropuerto y puerto

### ✨ Características Principales

| Característica | Descripción |
|----------------|-------------|
| 📤 **Envío masivo** | Límite configurable de 10 por ronda para evitar bloqueos |
| 🔄 **Personalización** | Reemplazo automático de `[Nombre]` en cada mensaje |
| 📎 **Adjuntos** | Envío automático de brochures de Google Drive |
| 📊 **Dashboard** | Interfaz en Google Sheets con tracking en tiempo real |
| ⏰ **Programación** | Envíos automáticos diarios (9am, 2pm, 6pm) |
| 🔐 **Sesión persistente** | Sin escanear QR en cada ejecución |
| 📈 **Historial** | Registro completo de todos los envíos |
